校内活動特設.

生きた探究と、地域へのまなざし。創英生が日常から広げる、社会との多様な結びつきをご紹介します。

社会と響き合う、創英の探究と実践

横浜創英では、教室の中にとどまらない多様なプロジェクトやボランティア活動が芽吹いています。
創英祭では、それぞれのグループが自発的に取り組んできた軌跡と、未来に向けた挑戦を特設展示としてご紹介いたします。

PROJECT 01 探究・まちづくり

YSTP

横浜創英タウンプロジェクト

地域社会や企業、自治体と深く連携しながら、横浜のまちづくりや社会的課題の解決に挑む探究プロジェクト。高校生の視点からまちの魅力を再発見し、新しい価値を創出します。

主な活動内容:
#地域連携 #企業コラボ #課題解決
詳細レポートを見る
PROJECT 02 地域貢献・交流

大口地域ボランティア

地域と共に歩むコミュニティ支援

学校の地元である大口商店街や地域イベントへ積極的に参加。世代を超えた心温まる交流を通じて、地域コミュニティの活性化と賑わいづくりに貢献しています。

主な活動内容:
#大口商店街 #祭事お手伝い #多世代交流
創英祭でのブース出展実施
PROJECT 03 環境改善・サステナビリティ

清掃ラボ

環境美化と空間プロデュース

校内や通学路、地域のクリーンアップ活動を科学的・探究的な視点からアプローチ。「きれいにする」ことの価値を問い直し、誰もが心地よく過ごせる環境づくりを目指します。

主な活動内容:
#環境美化 #学内美化 #サステナビリティ
