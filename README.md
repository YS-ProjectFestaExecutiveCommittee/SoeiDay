# வணக்கம்! <br>
நாங்கள் யோக்கோஹாமா சோய் மேல்நிலைப் பள்ளி "சோய் திருவிழா" (கலாச்சார விழா) நிர்வாகக் குழு. <br>
எங்கள் பள்ளியில் நடைபெறும் பல நிகழ்வுகளில், இது வரலாறும் பாரம்பரியமும் கொண்டது, மேலும் இது மிகவும் உற்சாகமானது! <br>
எங்கள் இணையதளத்தின் மூலம் இந்த நிகழ்வை நீங்கள் இன்னும் அதிகமாக ரசிப்பீர்கள் என்று நம்புகிறோம். <br>
இதை நீங்கள் ரசிப்பீர்கள் என்று நம்புகிறோம்! <br>
